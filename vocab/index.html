Semantic Notebook Vocabulary

Vocabulary for defining browser-based semantic teaching notebooks. A notebook is an ordered collection of labs, each occupying the full viewport. Each lab contains panels that are projections of a shared named graph.

Classes

sembook:Notebook

A named, ordered collection of Labs. Serialized as a JSON-LD file and fetched by the browser runtime on page load. The notebook is a first-class resource with its own IRI.

sembook:Lab

The fundamental teaching unit. A lab occupies the full viewport and is identified by a fragment identifier on the notebook URL. Each lab owns exactly one named graph in the quad store — the lab IRI is the named graph IRI.

sembook:Panel

Abstract base class for all panel types. A panel is a web component that is a projection of a lab's named graph, queried via a SPARQL query declared in its definition.

sembook:JsonLdPanel rdfs:subClassOf sembook:Panel

An interactive JSON-LD editor panel with a fetch affordance and a Parse button that commits the editor contents to the lab's named graph.

sembook:JsonLdContextPanel rdfs:subClassOf sembook:JsonLdPanel

A split-view JSON-LD editor: the document body and its @context are edited in two separate panes sharing one Parse button.

sembook:GraphPanel rdfs:subClassOf sembook:Panel

A Cytoscape.js graph visualization panel, rendering asserted (solid), inferred (dashed), and fetched (dotted) triples distinctly.

sembook:EntityPanel rdfs:subClassOf sembook:Panel

A list-based panel displaying named entities or vocabulary terms, filtered by the panel's SPARQL query.

sembook:VocabularyPanel rdfs:subClassOf sembook:Panel

A vocabulary explorer browsing class and property definitions and their hierarchy across a notebook's cumulative scope, distinguishing merely-used terms from described terms.

sembook:TabsPanel rdfs:subClassOf sembook:Panel

A layout container panel presenting its child panels as switchable tabs.

sembook:TurtlePanel rdfs:subClassOf sembook:Panel

A read-only Turtle serialization viewer for the panel's SPARQL-selected scope of the quad store.

sembook:TurtleWriterPanel rdfs:subClassOf sembook:Panel

An editable Turtle input panel, following the same Parse-button pattern as JsonLdPanel.

sembook:SparqlPanel rdfs:subClassOf sembook:Panel

An interactive SPARQL query editor panel with sample queries and a Run button that executes against the quad store without mutating it.

sembook:SparqlResultPanel rdfs:subClassOf sembook:Panel

Renders the result of a companion SparqlPanel's most recently executed query — Table, Turtle, or Graph views.

sembook:SampleQuery

A named, pre-authored SPARQL query offered in a SparqlPanel's sample-query dropdown.

sembook:Scope

The graph scope a panel projects: exactly the owning lab's named graph, or the cumulative union of every lab's graph up to and including the owning lab. Two named individuals: sembook:LocalScope and sembook:CumulativeScope.

sembook:LocalScope a sembook:Scope

The panel projects only the owning lab's named graph (plus its inferred companion).

sembook:CumulativeScope a sembook:Scope

The panel projects the union of every lab's named graph from the first lab up to and including the owning lab (each with its inferred companion) — how knowledge introduced in earlier labs stays visible in later ones.

Properties

sembook:title owl:DatatypeProperty · rdfs:subPropertyOf dcterms:title

The display title of a Notebook. Aligned to Dublin Core rather than reinvented — every sembook:title is a dcterms:title. domain: sembook:Notebook · range: xsd:string

sembook:labs owl:ObjectProperty

Ordered list of Labs in a Notebook. domain: sembook:Notebook · range: sembook:Lab

sembook:label owl:DatatypeProperty · rdfs:subPropertyOf rdfs:label

A human-readable display label for a Lab, Panel, or SampleQuery. Aligned to rdfs:label, so generic RDF tooling reads sembook display names for free. domain: Lab ∪ Panel ∪ SampleQuery · range: xsd:string

sembook:init owl:ObjectProperty

Seed data for a Lab, expressed as an embedded JSON-LD graph. domain: sembook:Lab · range: rdfs:Resource

sembook:panels owl:ObjectProperty

Ordered list of Panels within a Lab or TabsPanel. domain: Lab ∪ TabsPanel · range: sembook:Panel

sembook:cssClass owl:DatatypeProperty

Tailwind CSS utility classes applied directly to the rendered component element. domain: Lab ∪ Panel · range: xsd:string

sembook:sparql owl:DatatypeProperty

A SPARQL query string — on a Panel, determines what data it renders; on a SampleQuery, the query text offered as a starting point. domain: Panel ∪ SampleQuery · range: xsd:string

sembook:scope owl:ObjectProperty

The graph scope this panel projects — declared in the notebook definition so that scope is data, never inferred from a display label. When absent, the runtime default is panel-type-specific: GraphPanel → local; EntityPanel and VocabularyPanel → cumulative. domain: sembook:Panel · range: sembook:Scope

sembook:sampleQueries owl:ObjectProperty

Ordered list of SampleQuery entries offered in a SparqlPanel's dropdown. domain: sembook:SparqlPanel · range: sembook:SampleQuery

sembook:initialContent owl:DatatypeProperty

Seed text loaded into a panel's primary editing surface before the user makes any edits. domain: JsonLdPanel ∪ TurtleWriterPanel · range: xsd:string

sembook:initialContext owl:DatatypeProperty

Seed text loaded into a JsonLdContextPanel's separate @context editing pane. domain: sembook:JsonLdContextPanel · range: xsd:string
